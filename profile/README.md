## Tad's RNF

What is an RNF?  It is a random neural firing.  When I have an RNF, it 
generally results in the production of some half baked code, probably 
a circuit board or two, and occasionally a small bit of reflection about 
what I've learned.  

This is where the detritus that results lives.

I apologize if you end up dumber after visiting.
